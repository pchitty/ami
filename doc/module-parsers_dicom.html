Module: parsers/dicom

Dicom parser is a combination of utilities to get a VJS image from dicom files. scripts Relies on dcmjs, jquery, HTML5 fetch API, HTML5 promise API. image-JPEG2000 (jpx) is still in use, because Cornerstone does it and may have identified some edge corners. Ref: https://github.com/cornerstonejs/cornerstoneWADOImageLoader/blob/master/docs/Codecs.md https://github.com/cornerstonejs/cornerstoneWADOImageLoader/blob/a9b408f5562bde5543fc6986bd23fbac9d676562/src/shared/decoders/decodeJPEG2000.js#L127-L134
